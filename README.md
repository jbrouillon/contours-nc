# Contours NC

Site Quarto pour le blog **Contours NC** : sciences sociales, cartographie et dataviz en Nouvelle-Calédonie.

## Prévisualiser le site

Dans le terminal du projet :

```bash
quarto preview
```

## Générer le site

```bash
quarto render
```
